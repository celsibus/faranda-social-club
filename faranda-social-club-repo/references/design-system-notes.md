# Design System Notes — Faranda V4

## Fuente de referencia

**Faranda Homepage V4** — design system usado como guía obligatoria para la landing del Social Club.

---

## Principios visuales aplicados

### Tipografía
- **Sans-serif moderna** para textos principales
- Jerarquía clara: títulos grandes, cuerpo legible
- Espaciado generoso para respiración visual

### Paleta de color
- **Tonos neutros sofisticados** como base
- Acentos sutiles para CTAs
- Contraste alto para legibilidad

### Espaciado y grid
- Grid flexible responsive
- Espacios amplios entre secciones
- Alineación limpia y ordenada

### Tono visual
- **Aspiracional pero no artificial**
- Editorial/magazine-like
- Fotografía de alta calidad
- Minimalismo sofisticado

---

## Aplicación al Social Club

La landing del Faranda Social Club debe:
- Respetar la arquitectura visual de Faranda V4
- Pero tener personalidad propia dentro de esos límites
- Sentirse como una experiencia premium dentro del ecosistema Faranda

**Dirección aprobada:**
- Tono aspiracional/editorial
- NO estilo SaaS genérico
- NO excesivamente corporativo
- SÍ sofisticado, curado, con atención al detalle
