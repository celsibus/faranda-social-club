## Cómo trabajar con estas versiones

### Para revisar contenido
Abre `/social-club-v3/faranda-social-club-v3.html` en tu navegador.

### Para crear nueva versión
1. Copia la carpeta `/social-club-v3/`
2. Renombra como `/social-club-v4/` (o nombre descriptivo)
3. Haz tus cambios
4. Actualiza este README indicando la nueva versión
5. No borres versiones anteriores (son histórico)

### Para comparar versiones
Abre dos versiones en pestañas distintas del navegador y compara:
- Arquitectura de contenidos
- Copy
- Jerarquía visual
- Beneficios comunicados
