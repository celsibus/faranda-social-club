## Objetivo del proyecto

Crear una **membresía de pago** con:
- Identidad clara y diferenciación de marca
- Perks tangibles y operativamente viables
- Componente de pertenencia y experiencia social
- Venta directa incentivada
- Modelo económico rentable

**NO es un programa de puntos tradicional.**  
**NO es un loyalty program genérico.**
